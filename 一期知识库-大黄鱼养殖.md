## 二、适宜水质参数阈值表（核心规则库）

> 用于：水质告警阈值、大模型判断"水质是否适宜"的依据。分级为 适宜 / 警戒 / 危险。

| 指标 | 适宜范围 | 最适 | 警戒（关注） | 危险（干预/告警） | 单位 |
|------|---------|------|-------------|------------------|------|
| 水温 | 18–28 | 22–26 | <14 或 >30 | <8 或 >32 | ℃ |
| pH | 7.8–8.5 | 8.0–8.2 | <7.6 或 >8.8 | <7.0 或 >9.0 | — |
| 溶解氧 DO | ≥5 | ≥6 | 3–5 | <3（泛塘风险） | mg/L |
| 溶解氧饱和度 | ≥80 | ≥90 | 50–80 | <50 | % |
| 盐度 | 25–32 | 28–32 | <20 或 >35 | <15 或 >38 | ppt(‰) |
| 电导率 | 参考对应盐度 | — | — | — | mS/cm |
| 氨氮（NH₃-N） | <0.5 | <0.2 | 0.5–1.0 | >1.0 | mg/L |
| 亚硝酸盐（NO₂⁻） | <0.1 | <0.05 | 0.1–0.2 | >0.2 | mg/L |
| 浊度/透明度 | 透明度 ≥30cm | — | <20cm | — | cm |

**校准建议**：DO 在夜间至黎明最低，应以日最低值为告警判据；盐度受降雨/径流影响，大雨后需重点看。

> 📌 对应平台现状：连江渔排 pH=6.27、DO=3.9 mg/L 均已进入**危险区**（pH<7.0、DO<3），但与同区域其他站点（pH≈8、DO 7–8）严重背离，**判定为传感器故障或局部异常**，需现场校准而非直接按阈值告警。

## 三、投喂经验表（投喂率 = 占鱼体重 %）

### 3.1 按鱼体重阶段的基础日投饲率

| 鱼体重阶段 | 日投饲率（占体重%） | 投喂频次 |
|-----------|-------------------|---------|
| 稚鱼 <10 g | 4–6 | 3–4 次/日 |
| 幼鱼 10–150 g | 2–4 | 2–3 次/日 |
| 中成鱼 150–500 g | 1–2 | 2 次/日（早、傍晚） |
| 亲鱼/越冬 >500 g | 0.5–1 | 1–2 次/日 |

### 3.2 按水温的投饲率修正系数

| 水温 | 投喂策略 | 修正 |
|------|---------|------|
| 22–26℃（最适） | 正常投喂 | ×1.0 |
| 18–22 / 26–28℃ | 正常偏保守 | ×0.8 |
| 14–18℃ | 减量 | ×0.5 |
| 28–30℃ | 减量、避开正午 | ×0.5–0.7 |
| <14 或 >30℃ | 大幅减量或停喂 | ×0.2–0（停喂） |
| <8 或 >32℃ | 停喂、保命 | 0 |

**最终日投喂量 = 鱼体总重 × 基础投饲率 × 水温修正系数**，再按 DO/天气/摄食响应（声纳）动态下调。

### 3.3 投喂调整规则
- **DO <4 mg/L**：暂停投喂，待回升后再投；夜间低氧期不投喂。
- **高温（>30℃）**：避开 10:00–15:00，早晚投喂。
- **雷雨/台风/闷热低气压**：减量 30–50% 或停喂。
- **摄食响应差**（声纳：抢食不积极、残饵多）：下次下调 10–20%。
- **大潮期/急流**：适当减量，避免饲料流失。

### 3.4 饲料营养与饲料系数 FCR
- 适宜粗蛋白：**幼鱼 45–52%**，成鱼 ≥40%
- 常见 FCR（饲料系数）：**1.5–2.5**（鲜杂鱼饵料偏高，配合饲料偏低）
- 目标 FCR 越低越经济；FCR 异常升高提示消化不良/水温不适/病害

## 四、体长–体重关系（生物量换算核心）

$$W = a \cdot L^{b}$$

| 参数 | 值 | 说明 |
|------|----|------|
| a | **0.024** | 条件因子 |
| b | **2.848** | 近等速生长（b≈3） |
| L 单位 | cm | 体长 |
| W 单位 | g | 体重 |

**来源**：养殖群体（三都湾/选育 F₂），体长范围约 20–359 mm / 19–52 cm。
**用法**：声纳测得体长 L → 单尾均重 W；**生物量 = 声纳计数 N × 均重 W**。

> 校准建议：不同选育品系 a 值有差异，建议本平台按实测 (L,W) 数据对回归标定。

## 五、生长参数（von Bertalanffy，二期生长预测用）

**体长生长方程**：$L_t = L_\infty[1 - e^{-k(t-t_0)}]$

| 参数 | 值 | 说明 |
|------|----|------|
| L∞（渐近体长） | 385.4 mm | 官井洋群体（叶金清等 2012，ELEFAN） |
| k（生长系数） | 0.43 /年 | |
| t₀ | −0.32 年 | |
| 生长拐点年龄 | ≈2.2 龄 | |

> 注意：此为**野生/资源评估群体**参数，养殖群体生长更快、拐点更早；二期应以本平台声纳体长时序重新拟合养殖群体参数，不宜直接套用。

## 六、告警规则总结（可直接接入规则引擎）

| 规则 | 条件 | 动作 |
|------|------|------|
| 低氧告警 | DO<3 mg/L 或饱和度<50% | 立即增氧、停喂 |
| 低氧预警 | DO 3–5 mg/L | 关注、减喂、准备增氧 |
| 水温过高 | 水温>30℃ | 减喂、避高温、防泛塘 |
| 水温过低 | 水温<14℃ | 大幅减喂 |
| pH 异常 | pH<7.6 或>8.8 | 排查水质/校准传感器 |
| 盐度波动 | 日变>3 ppt 或超 25–32 | 检查淡水注入/换水 |
| 传感器背离 | 单站水质与邻站系统性背离 | 疑似设备故障，校准 |
| 设备离线 | 传感器离线 | 现场维护 |
| 台风/大浪预警 | 气象预警 | 加固网衣、必要时提前起捕 |
| 死亡率上升 | 死鱼数环比异常升 | 停料、巡检、用药上报 |

## 八、知识库使用方式

1. **注入大模型（一期）**：把第二章阈值表 + 第三章投喂表 + 第六节规则，作为 system prompt 的"专家知识"注入；每站点实时数据作为 user 输入，让大模型据此给投喂/养殖建议。
2. **规则引擎（一期）**：第二、三、六节直接转成阈值判断，生成水质告警与投喂基线建议。
3. **小模型特征/标签（二期）**：第四节体长体重、第五节生长参数作为生物量/生长模型的先验；阈值作为风险模型的标签来源。
4. **持续校准**：所有参数标记"参考值"，随本平台实测数据积累持续更新版本。
