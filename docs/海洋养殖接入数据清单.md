## 二、已接入数据源分类（共 8 大类）

### 数据源 2：水质环境实时监测（核心养殖数据）

- **接口**：`GET /api/biz/workspace/environment/sites?page=1&pageSize=100`
- **历史趋势**：`GET /api/biz/workspace/environment/sites/{siteId}/history`（需带时间范围参数）
- **采集指标**（7 项水质参数）：

| 字段 | 含义 | 单位 |
|------|------|------|
| `temperature` | 水温 | °C |
| `ph` | 酸碱度 | pH |
| `dissolvedOxygenSaturation` | 溶解氧饱和度 | % |
| `dissolvedOxygenConcentration` | 溶解氧浓度 | mg/L |
| `salinity` | 海水盐度 | ppt |
| `conductivity` | 电导率 | mS/cm |
| `tds` | 溶解性总固体 | g/L |

### 数据源 5：海洋气象数据（实时 + 预报）

- **接口**：`GET /api/biz/workspace/sites/{siteId}/weather`
- **数据来源**：和风天气（QWeather，依据 icon 编码体系判断）
- **包含内容**：
  - `now`：实时天气（气温、体感、天气现象、风向风力、湿度、降水量、气压、能见度）
  - `daily`：3 天预报（今天/明天/后天，白天夜间天气、最高最低温、风力、湿度、降水）

### 数据源 8：AI 视觉识别 + 告警记录

#### 8.1 AI 识别（鱼类计数 / 鱼体生长 / 产量分析）

- **汇总接口**：`GET /api/biz/workspace/ai-analysis/sites/{siteId}/summary`
- **总览识别影像**：`/api/biz/workspace/overview` 的 `recognitions` 字段
- **识别图片服务**：`http://<平台地址>:9920/outputs/{uuid}.png`
- **AI 能力**（summary 结构）：
  - `algorithmStreams`：算法流
  - `latestRecognitions` / `historyRecognitions`：识别记录
  - `growthSummary`：鱼体生长分析（平均/最大/最小**体长** cm）
  - `productionSummary`：产量分析（体长 cm + **体重** g）

**总览识别影像样例**（来源：闽江大学实验室，摄像头 `34020000001320000001`，网关 `mju-gw-01`）：

```json

  {"id":"2205","imageUrl":"http://<平台地址>:9920/outputs/6ea1887379984fa2a9ffefec72bc19f6.png", "name":"识别到 6 尾鱼", "fishCount":6, "time":"05-30 23:41"},
```

## 五、迭代建议（基于数据现状）

1. **水质历史趋势接口**（`environment/sites/{id}/history`）目前直接 GET 返回系统异常，建议补全参数定义或改为 POST，让前端能画趋势曲线。
2. **AI 生长/产量数据缺失**：growthSummary/productionSummary 均为 unknown，但已识别 11964 次。建议把鱼体长度/体重识别结果落库，激活生长曲线与产量预估。
3. **告警分页接口**未在前端暴露稳定 URL，建议补充 `/warning/page` 列表接口，支持按站点/级别/时间筛选。
4. **数据完整性**：连江渔排实时数据 pH/溶解氧明显异常（可能是传感器故障），可结合设备离线状态做数据质量校验。
5. **可拓展接入**：台风路径、海洋水文（流速/浪高）、投饵机/增氧机控制状态、饵料库存、养殖品种与放苗/起捕记录等业务数据当前未见，可作为下一期迭代的数据源。
