# 声纳数据接入 —— 设备选型备忘

> 编制日期：2026-07-15
> 用途：智慧海洋养殖管控平台下一期数据源接入（声纳）

## 选型理由

1. 成像声纳是浊水（福建近海）下光学摄像头的必要补充，一台覆盖计数/巡检/测长三大需求。
2. 价格精准落在 20–40 万预算区间；同档进口（BlueView/Oculus）顶预算上限，旗舰 ARIS 3000 约 60–85 万超预算。
3. 国产厂商 SDK/接口开放，便于二次开发接入管控平台（与现有 MQTT 网关、GB2818 摄像头同思路接入新设备类型）。
4. 生物量采用"声纳图像测长 → 体长体重经验公式换算"，满足网箱养殖实用需求；如二期需精确 TS 目标强度，可再叠加分裂波束（BioSonics DT-X / Simrad EK80）。
