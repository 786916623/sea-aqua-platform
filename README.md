# 智慧海洋养殖管控平台

福建海域网箱养殖管控平台 —— 需求分析、HTML 原型、知识库、功能清单与报价。

## 目录结构

- `prototype/` — HTML 高保真原型（12 个页面 + 导航页）
  - `index.html` 总览驾驶舱
  - `water-quality.html` 水质监测
  - `sites.html` 养殖站点
  - `feeding.html` 投喂管理
  - `biomass.html` 鱼群生物量（声纳）
  - `weather.html` 气象海洋
  - `alerts.html` 告警中心
  - `reports.html` 报表中心
  - `ai-advice.html` AI 智能分析
  - `ai-chat.html` AI 智能问答
  - `agent.html` 智能体配置
  - `knowledge.html` 知识库管理
  - `导航.html` 原型导航
  - `assets/` 共享样式与脚本（style.css / nav.js）
- `docs/鱼群养殖预测分析-需求分析文档.md` — 需求分析文档
- `docs/功能清单与报价.md` — 功能清单与工时报价
- `docs/一期知识库-大黄鱼养殖.md` — 大黄鱼养殖知识库
- `docs/声纳选型备忘.md` — 声纳选型
- `docs/海洋养殖接入数据清单.md` — 已接入数据清单

## 预览

浏览器打开 `prototype/导航.html` 即可浏览全部原型页面。

## 说明

- 原型为静态 HTML + ECharts（CDN），双击即可运行。
- 文档与原型对齐，可作为需求/交付/报价依据。
